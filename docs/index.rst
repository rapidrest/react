===============================================================================
@rapidrest/react
===============================================================================

A library for serving React pages from a `RapidREST <https://github.com/rapidrest>`_
server, with Next.js-style file-based routing, server-side rendering, optional
client-side hydration, and browser live-reload in development.

Repository: ``git clone https://github.com/rapidrest/react.git``
